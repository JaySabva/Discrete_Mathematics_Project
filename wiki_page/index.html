Mathematics of Elliptic curves on finite fields :

Elliptic Curves:

  Elliptic curves are curves described by the equation y2 = x3 + ax + b over an algebric field K . Where a,b ∈ K. It looks something like this:

Point at Infinity:

  The point at infinity is denoted by the symbol O. It is the identity element / point of elliptic curve maths.

O + O = O

P + O = P

Here, P + P + O = O

Point Negation:

For Point P(x, y) on the Elliptic curve Q(x, -y) is defined as negation of P. Their sum results into the identity point.

Point Addition:

  For Points P and Q on the curve if the line passing from them intersects the curve at R then P + Q + R = 0. Here, S is negation of R ⇛ S + R = 0 ⇛ S = P + Q. Mathematically,

if m = (yq - yp)/(xq - xp) , then...

xs = m2 - xq - xp

ys = m(xp - xs) - yp

Point Doubling:

  If tangent at P intersect the curve at Q then according to the addition rule P + P + Q = 0 ⇛ 2P + Q = 0. if S is negation point of Q then, S = 2P. Mathematically,

Here, m = y'(x)(xp, yp) , then...

xs = m2 - 2xp

ys = m(xp - xs) - yp

Point Scalar Multiplication:

  We can multiply a point with a number k by using Doubling and Addition concepts. This algorithm generally takes log2k time to compute M = k*P.

if S= 2P, then...

3P = 2P + P = S + P

4P = 2S = S + S and so on...

Elliptic Curves over finite fields:

  For a prime number p, We can construct a finite field Zp with p number of elements. We can define a curve over Field Fp = Zp ⊗ Zp as y2 ≡ x3 + ax + b (mod p). Here, a,b ∈ Zp. The order of the curve n is the total number of finite field pairs that lie on this curve, including the point at infinity O.
  If we add any two points or multiply a point by a number in a cyclic subgroup, the result will be within that same cyclic subgroup. There is no overlap between any two cyclic subgroups. If each cyclic subgroup contains r points, then the cofactor of the curve is h = n/r. It is better to have less cofactor for security efficiency ideally h should be 1 ⇛ All points on the curve should be in a single cyclic group.
